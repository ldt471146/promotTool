# AI 提示词优化器 v0.1

基于瑞士国际风格设计的提示词优化工具。

## 快速启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173

## 核心功能

1. **提示词优化** - 将模糊需求转化为结构化提示词
2. **模板库** - 内置编程/写作/翻译模板
3. **历史记录** - 保存和复用优化结果

## 设计特点

- 瑞士国际风格（Swiss International Style）
- 严格网格系统
- 无衬线字体
- 黑白配色 + 红色点缀
- 硬边缘阴影效果
- 响应式设计

## 技术栈

- 原生 JavaScript（无框架）
- Vite（开发服务器）
- Tailwind CSS（样式）
